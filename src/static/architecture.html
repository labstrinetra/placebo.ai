System Architecture

How Placebo AI Ensures Factual Medical Responses through a Multi-Layered Pipeline.

Layer 1: Multimodal Ingestion

Extracts structured medical knowledge from unstructured PDFs and textbooks. Our custom Vision-Language Model is optimized for medical layouts, detecting text, tables, chemical formulas, and anatomical diagrams while preserving their semantic relationships.

Layer 2: Vector Intelligence

Converts medical data into 384-dimensional semantic vectors. Using FAISS indexing, we enable sub-millisecond retrieval of the most relevant knowledge chunks based on meaning, not just keywords.

Layer 3: Grounded RAG Layer

The heart of zero-hallucination AI. Generates answers strictly from retrieved context. Every statement is required to reference a source chunk, preventing the model from using training data.

Layer 4: Structured Output

Formats responses for professional medical use. Includes human-readable answers, structured JSON for API integration, and markdown reports with full attribution.

Architecture Principles

  • 01
    Retrieval First, Generation Second We invert the typical LLM flow by prioritizing evidence retrieval as the anchor of the process.
  • 02
    Constraint-Based Generation Strict system prompts eliminate the degrees of freedom that allow hallucinations.
  • 03
    Transparency Through Citation Every factual claim is linked to a source page, making the system auditable and trustworthy.
